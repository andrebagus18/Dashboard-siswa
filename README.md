# 🎓 Student Dashboard with PHP Native

## 🚀 Features

- ➕ Add student data
- 📋 View student list
- ✏️ Edit student information
- ❌ Delete student data
- 💾 Data stored using LocalStorage
- 🎨 Responsive and clean UI design
- ⚡ Lightweight (no database required)

---

## 🛠️ Tech Stack

- PHP Native (basic project structure)
- JavaScript (ES6 DOM manipulation & logic)
- CSS3 (styling & responsive UI)
- HTML5 (structure)
- LocalStorage (client-side storage)

---

## 📦 How to Run This Project

Clone repository:

bash
git clone <repository-url>
cd dashboard-siswa

Start local server:

Using XAMPP:

txt
Move project to:
htdocs/dashboard-siswa

Open:

txt
http://localhost/dashboard-siswa

Or use PHP server:

bash
php -S localhost:8000

Then visit:

txt
http://localhost:8000
